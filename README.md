# Smart Calculator (Java)

This is a simple calculator project built using Java.

## 🔹 Features

* Addition
* Subtraction
* Multiplication
* Division
* User input using Scanner
* Loop for continuous calculations

## 🔹 Technologies Used

* Java
* VS Code / IntelliJ

## 🔹 How to Run

1. Compile the program:
   javac SmartCalcy.java

2. Run the program:
   java SmartCalcy
